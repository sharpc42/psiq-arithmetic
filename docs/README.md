# psiq-arithemtic

Implementing arithmetic algorithms in PsiQuantum Workbench as part of the EE 522 mentorship program at the University of Washington.

The currently implemented algorithms are:

**Addition**
- [TTK](https://arxiv.org/pdf/0910.2530) (in-place)
- [CT](https://www.researchgate.net/publication/3385042_Quantum_full_adder_and_subtractor) (out-of-place)
- [Wang](https://doi.org/10.1007/s11432-015-5411-x) (out-of-place)
- [Gayathri](https://www.researchgate.net/publication/352932278_T-Count_Optimized_Wallace_Tree_Integer_Multiplier_for_Quantum_Computing) (out-of-place)

**Multiplication**
- [Schoolbook](https://github.com/Strilanc/quantum-karatsuba-2019/tree/master) (with TTK)
- [JHHA](https://arxiv.org/abs/1608.01228)

The authors are Kush Aggarwal, Thomas Huang, and Christopher Sharp. The industry mentor is Mariia Mykhailova with PsiQuantum.
